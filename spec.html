Neo-Brutalism
01
Обзор системы

WF Catalyst использует жёсткую, высококонтрастную, модульную визуальную систему:

  • Чёрные бордеры 3px на всём
  • Плоские заливки, без градиентов
  • Блочная композиция с чёткими секциями/карточками
  • Геометрические заголовки верхним регистром (Neutro Bold)
  • Синий #198ED5 — основной сигнал бренда
  • Жёлтый #FCD84D — цвет CTA
  • Тёмная/светлая темы с одинаковыми правилами
  • border-radius: 0 на всех блоках (pill только на тегах/бейджах)

Стиль: прямой, смелый, чистый, операционный — немного агрессивный в хорошем смысле.

08
Do / Don't

✅ DO

  • Плоские заливки
  • Чёрные бордеры 3px на всём
  • border-radius: 0
  • Жёсткая тень 6px 6px 0
  • UPPERCASE Neutro для заголовков
  • Цвета строго по токенам
  • Высокий контраст ≥ 4.5:1
  • Модульная сетка

❌ DON'T

  • Градиенты
  • Мягкие тени / drop shadow / glow
  • Скруглённые углы (кроме pill)
  • Тонкие серые бордеры
  • Пастельные макеты без чёрной структуры
  • Glassmorphism
  • Rounded friendly SaaS-стиль
  • Много акцентных цветов в одном блоке

Паттерны макетов

Hero Section: Белая карточка с 3px чёрным бордером. Слева: заголовок (Neutro) + текст (Inter) + CTA. Справа: чёрный фон с перекрывающим блоком (белый или синий #93D1F7).

Two-Column Card: Равные колонки с бордером. Опционально: тёмная левая с белым текстом.

Full-Dark Card: Фон #000, текст #FFF, графика с синей шкалой, жёлтый CTA с тенью.

10
Принципы дизайна
  1. Neo-brutalism: Жёсткие чёрные бордеры на всём. Без мягких теней, градиентов, скруглений (кроме pill).
  2. Высокий контраст: Чёрные бордеры #000 по 3px на всех карточках, кнопках и блоках.
  3. Жёсткие тени: Кнопки и ключевые элементы — сплошной чёрный прямоугольник (~6px, без blur).
  4. Смелая типографика: Neutro Bold для всех заголовков — uppercase, геометрический, индустриальный.
  5. Чистый body-текст: Inter для всего основного текста — современный, читабельный, нейтральный.
  6. Палитра на базе синего: Основная визуальная идентичность — #198ED5 с 5-ступенчатой шкалой.
  7. Жёлтый CTA: Основной call-to-action всегда жёлтый #FCD84D с чёрным текстом и тенью.
  8. Акценты умеренно: Оранжевый, жёлтый, зелёный — только семантика, ≤10% площади.
  9. Две темы: Полная поддержка light/dark с чёткими переключениями токенов.
  10. Геометрические макеты: Карточки, блоки и секции — прямоугольные, строгая сетка.

Компактный промпт (AI / quick-ref)
WF Catalyst visual style = bold modular system with heavy black borders (3px), flat colors, zero radius, Neutro 700 for headings, Inter 400/600 for body, primary blue #198ED5, support blues #93D1F7 and #C4E8FE, dark background #282828, light background #EFF5F9, white cards, black structural blocks, yellow CTA #FCD84D, semantic green #00B176, semantic red #FD4E2C. Hard shadow offsets on buttons (solid black, ~6px). No soft gradients, no rounded SaaS look, no weak gray outlines, no glassmorphism.
