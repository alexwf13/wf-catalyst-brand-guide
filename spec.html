Neo-Brutalism
01
Обзор системы

WF Catalyst использует жёсткую, высококонтрастную, модульную визуальную систему:

  • Чёрные бордеры 3px на всём — всегда чёрные, никогда белые
  • Плоские заливки, без градиентов
  • Блочная композиция с чёткими секциями/карточками
  • Геометрические заголовки верхним регистром (Neutro Bold / Resolve Sans Black Ext для кириллицы)
  • #93D1F7 (brand-400) — основной цвет поверхностей (карточки, панели)
  • Жёлтый #FCD84D — цвет CTA
  • Тень: влево-вниз -6px 6px 0
  • Тёмная/светлая темы с одинаковыми правилами

Стиль: прямой, смелый, чистый, операционный — немного агрессивный в хорошем смысле.

08
Do / Don't

✅ DO

  • Плоские заливки
  • Чёрные бордеры 3px на всём — никогда белые
  • border-radius: 0
  • Жёсткая тень -6px 6px 0 (влево-вниз)
  • UPPERCASE Neutro / Resolve Sans для заголовков
  • brand-400 (#93D1F7) для карточек/панелей
  • Цвета строго по токенам, с hex-кодами
  • Высокий контраст ≥ 4.5:1
  • Модульная сетка

❌ DON'T

  • Градиенты
  • Мягкие тени / drop shadow / glow
  • Скруглённые углы (кроме pill)
  • Тонкие серые бордеры
  • Белые бордеры — никогда
  • Голубые полоски / декоративные линии
  • Пастельные макеты без чёрной структуры
  • Glassmorphism
  • Rounded friendly SaaS-стиль
  • Хелпер-синие (700/900/200) для больших поверхностей
  • Белый текст на синих тегах

Паттерны макетов

Hero Section: Белая карточка с 3px чёрным бордером. Слева: заголовок (Neutro) + текст (Inter) + CTA. Справа: чёрный фон с перекрывающим блоком (белый или синий #93D1F7).

Two-Column Card: Равные колонки с бордером. Опционально: тёмная левая с белым текстом.

Full-Dark Card: Фон #000, текст #FFF, графика с синей шкалой, жёлтый CTA с тенью.

10
Принципы дизайна
  1. Neo-brutalism: Жёсткие чёрные бордеры на всём. Без мягких теней, градиентов, скруглений (кроме pill).
  2. Чёрные бордеры всегда: #000 по 3px на всех карточках, кнопках, тегах. Белые бордеры — никогда.
  3. Тень влево-вниз: Кнопки — сплошной чёрный прямоугольник -6px 6px 0 (без blur).
  4. Смелая типографика: Neutro Bold (Latin) / Resolve Sans Black Ext (Cyrillic) — uppercase, геометрический.
  5. Чистый body-текст: Inter для всего основного текста — современный, читабельный, нейтральный.
  6. brand-400 — основной: #93D1F7 — цвет поверхностей. Остальные синие — равнозначные хелперы для тегов.
  7. Жёлтый CTA: Основной call-to-action — #FCD84D с чёрным текстом и тенью.
  8. Акценты умеренно: Жёлтый #FCD84D, зелёный #00B176, красный #FD4E2C — только семантика, ≤10%.
  9. Две темы: Light/dark с одинаковыми структурными правилами. Бордеры чёрные в обеих.
  10. Геометрические макеты: Карточки, блоки и секции — прямоугольные, строгая сетка.
11
Компактный промпт (AI / quick-ref)
WF Catalyst visual style = bold modular system with heavy black borders (3px, always black, never white), flat colors, zero radius, Neutro Bold 700 (Latin) / Resolve Sans Black Ext (Cyrillic) for headings, Inter 400/600 for body. Primary surface color #93D1F7 (brand-400) for cards/panels. Helper blues #198ED5, #0265A1, #C4E8FE — equal-level, only for tags/highlights. Page background #EFF5F9 (not white). Dark background #282828. Yellow CTA #FCD84D with shadow -6px 6px 0 #000. Semantic green #00B176, red #FD4E2C. Tags: 3px border, black text always. No white borders, no blue stripes, no soft gradients, no rounded SaaS look, no glassmorphism.
