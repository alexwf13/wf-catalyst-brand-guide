04

Usage Rules

Panels & Blocks
  • Main panel: #FFFFFF
  • Accent panel: #93D1F7
  • Helper panels: #C4E8FE, #198ED5, #0265A1
  • Helpers ≤ 10% of total surface area
  • Can extend in 10% opacity steps
Borders
  • Color: #000000
  • Width: 3px
  • Radius: 0px — always
  • No soft shadows, no glassmorphism
  • Hard edges define the system
Color Logic
  • Primary site color: #198ED5
  • Blue = strongest brand signal
  • All palette is built around this blue
  • Yellow is CTA-only, not decorative
  • Semantic green/red only for states
Semantic Colors
  • Success: #00B176
  • Warning: #FCD84D
  • Error: #FD4E2C
  • Don't mix into charts unless meaning is success/failure
05

Style Behavior

✓ Do
  • Direct, structured, confident
  • High contrast, bold layout
  • Strong black borders everywhere
  • Modular block composition
  • Flat fills, zero radius
  • Modern and a little aggressive
✕ Don't
  • Pastel-heavy without black structure
  • Rounded, friendly SaaS-style UI
  • Subtle gray borders
  • Soft shadows and glassmorphism
  • Gradient-heavy treatments
  • Too many accent colors in one block
06

Composition & Hierarchy

Contrast Hierarchy

1 Structure Black / White
2 Brand Emphasis #198ED5
3 Secondary Emphasis #93D1F7 / #C4E8FE
4 Action (CTA) #FCD84D
5 Semantic State Green / Red only when state matters

Chart Color Progression

Neutral → Strong. Don't mix semantic red/green into charts unless meaning is explicitly success/failure.

Card & Section Rules

  • — Use strong rectangular blocks
  • — Separate content into obvious modules
  • — Prefer: big title + supporting block + small accent element
  • — White card on dark area, dark card on light area, blue accent for emphasis
07

Buttons

Button Spec
  • Background: #FCD84D
  • Text: #000000
  • Border: 3px solid #000
  • Radius: 0px
  • Label: uppercase, bold, compact
  • Shadow: black offset underlay

Short Brand Prompt (for AI / Designers)

Wonderflow visual style = bold modular system with heavy black borders, flat colors, zero radius, Neutro for headings, Inter for body, primary blue #198ED5, support blues #93D1F7 and #C4E8FE, dark background #282828, light background #EFF5F9, white cards, black structural blocks, yellow CTA #FCD84D, semantic green #00B176, semantic red #FD4E2C. No soft gradients, no rounded SaaS look, no weak gray outlines.
